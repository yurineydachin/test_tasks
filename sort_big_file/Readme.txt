Задачка достаточно простая.

Входные данные:
Будем считать что у нас есть компьютер, на котором объем оперативной памяти составляет 2 мегабайта памяти.

Есть файл размером 2 гигабайта, в котором построчно забиты неупорядоченно числа
Например:

25908
34
3463463
345
345345
37687978
и тд

Числа могут дублироваться.

Задача:
Необходимо реализовать алгоритм сортировки числ в данном файле от меньшего к большему, построчно.
Нельзя использовать никакие сторонние библиотеки и фреймворки, только стандартный php.
Алгоритм должен реализован по канонам ООП, с использованием классов, интерфейсов/абстрактных классов.

Необходимо учитывать входные данные, что в память может поместиться на более 2 мегабайт данных.
